# 에이전트 역할 요약 가이드 (Agent Roles Summary)

이 문서는 `agent_prompt/` 폴더 내의 각 프롬프트 파일을 분석하여 정의된 에이전트들의 핵심 역할을 한 줄로 요약한 가이드입니다.

| 에이전트 ID | 역할 명칭 | 핵심 요약 (Description) |
| :--- | :--- | :--- |
| **0** | 수석 AI 윤리 및 법률 전략가 | 글로벌 규제 준수 및 AI 윤리 가드레일을 설계하여 지속 가능한 혁신 모델 제안 |
| **1** | 수석 AI & 로보틱스 엔지니어 | 하드웨어와 지능이 결합된 물리적 자동화 및 로보틱스 비즈니스 모델 설계 |
| **2** | 수석 행동과학자 및 심리 전략가 | 인간의 행동 패턴과 인지 편향을 분석하여 리텐션이 높은 심리적 서비스 설계 |
| **3** | 실리콘밸리 시니어 심사역 | 압도적인 시장 지배력과 수익 구조를 가진 파괴적인 유니콘 비즈니스 모델 발굴 |
| **4** | 수석 핀테크 및 블록체인 전략가 | 자산 토큰화 및 금융 프로토콜을 통한 가치 전달 체계의 혁명적 개선 설계 |
| **5** | 수석 SCM 및 물류 혁신 전략가 | AI와 IoT를 결합하여 공급망 비효율을 제거하고 최적화된 글로벌 유통 네트워크 설계 |
| **6** | 수석 에듀테크 전략가 | 인지과학 기반의 적응형 학습 경로를 제공하는 차세대 교육 플랫폼 모델 제안 |
| **7** | 수석 크리에이티브 디렉터 | 기술과 감성을 연결하여 강력한 팬덤을 형성하는 몰입형 미디어 및 콘텐츠 생태계 설계 |
| **8** | 수석 스마트 시티 전략가 | 도시 인프라와 데이터를 결합하여 운영 효율을 극대화하는 지능형 도시 운영체제 설계 |
| **9** | 수석 사이버 보안 전략가 | 제로 트러스트 기반의 최첨단 보안 체계와 비즈니스 회복 탄력성 설계 |
| **10** | 수석 소프트웨어 문제 해결사 | 복잡한 현실 문제를 최적의 알고리즘과 자동화 프로세스로 해결하는 실무 솔루션 제안 |
| **11** | 수석 데이터 사이언티스트 및 AI 모델러 | 방대한 데이터 속에서 비즈니스 가치를 창출하고 머신러닝/딥러닝 기반의 ‘데이터 플라이휠’ 설계 |
| **12** | 수석 커리어 코치 및 진로 설계자 | 대학생들의 막막한 진로 고민을 공감과 분석을 통해 시장 트렌드에 맞는 구체적인 실행 계획으로 설계 |
| **A** | 수석 디지털 제품 아키텍트 | 추상적 아이디어를 실행 가능한 시스템 구조와 기술 스택으로 설계하는 기술 설계자 |
| **B** | 수석 리스크 분석가 및 레드팀 디렉터 | 아이디어의 치명적 결함을 찾아내고 현실적인 시장 검증과 리스크 분석을 통해 비즈니스 모델 강화 |
| **C** | 수석 비즈니스 분석가 | 전문가 간의 대화를 분석하여 구조화된 사업 기획서와 데이터(JSON)로 최종 요약 |
| **R** | 수석 정보 설계자 및 대화 분석가 | 복잡한 대화 로그를 분석하여 즉각적인 이해와 데이터베이스화가 가능한 구조적 요약 보고서 및 JSON 생성 |
